# Guardador de Contraseñas 3000

Un pequeño proyecto en Python para practicar la lectura y escritura de archivos desde la terminal.

## ¿Cómo funciona?
- `contra.py`: Sirve para ingresar nuevos usuarios y contraseñas.


## Uso
Para ejecutar el guardador:
`python contra.py`
